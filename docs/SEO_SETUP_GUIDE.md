# Post-Deployment SEO Checklist

## Google Search Console
1. Go to `https://search.google.com/search-console`
2. Add property: `https://enrollurm.com`
3. Verify via DNS TXT record (recommended) or HTML file
4. Submit sitemap: `https://enrollurm.com/sitemap.xml`
5. Request indexing for homepage immediately after launch
6. Monitor Coverage report for crawl errors weekly

## Google Analytics 4
1. Create GA4 property at `https://analytics.google.com`
2. Add measurement ID to app environment variables
3. Verify real-time data is flowing after deployment
4. Set up conversion events:
   - `contact_form_submit`
   - `counselor_whatsapp_click`
   - `counselor_email_click`
   - `book_call_click`
   - `enrollment_quiz_complete`

## Bing Webmaster Tools
1. Go to `https://bing.com/webmasters`
2. Import from Google Search Console
3. Submit sitemap

## Social Verification
1. Add Facebook domain verification meta tag
2. Add Pinterest verification tag if needed
3. Validate Twitter Card at `https://cards-dev.twitter.com/validator`

## Backlink Strategy (Post-Launch)
Priority sources to pursue:
- DAAD listing opportunities
- uni-assist.de partner mentions
- Arab student community forums and groups
- Study abroad blogs and YouTube channels
- German embassy websites in Arab countries
- University partner pages
